# Change Log

All notable changes to the "WebDAV Workspace" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2024

### Added
- Improved sync functionality
- Enhanced error handling

### Fixed
- Various bug fixes and stability improvements

## [1.0.0] - 2023-01

### Added
- Initial release of the WebDAV FileSystemProvider
- Support for Basic and Digest authentication
- Secure credential storage using VS Code SecretStorage
- Mount WebDAV folders as VS Code workspaces
- Direct file editing on remote WebDAV servers
- Bidirectional sync between local and remote folders
- Auto-sync on file changes
- Manual sync triggers
- Multiple sync configurations support
- Sync status tracking and logging
- Dedicated sync view in activity bar
- Setup wizard for easy configuration
- WebDAV browser for remote folder navigation
